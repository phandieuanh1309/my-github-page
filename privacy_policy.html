We only access and use publicly available data from the TikTok Shop API. No personal data is collected, stored, or shared. We retrieve public product information, including product names, prices, sales data, and reviews.

We use the data collected from the TikTok Shop API to:

We take security seriously: we do not store any user personal data on our servers. We comply with TikTok’s Developer Policies and Data Security Requirements.

4. Third-Party Services

This application does not share or sell any data to third-party services. We only use TikTok’s official API, and all data remains within TikTok’s ecosystem.

5. User Rights & Control

Users have the right to revoke API access at any time via their TikTok settings.
